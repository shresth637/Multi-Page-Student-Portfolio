# Multi-Page-Student-Portfolio
Problem Statement: Multi-Page Student
Portfolio Website
Objective:
Design and develop a multi-page Personal Student Portfolio Website using only 
HTML and Basic CSS. The website must consist of the following four separate 
HTML pages:
• index.html → Home Page
• about.html → About Me Page
• skills.html → Skills & Projects Page
• contact.html → Contact Form Page
Problem Description:
The website must clearly demonstrate correct usage of:
• HTML structure and semantic layout
• CSS styling
• Float for layout
• Flexbox for layout
• HTML Forms
• Proper folder structure
• GitHub repository hosting
Functional Requirements:
NAVIGATION BAR
A common navigation bar must appear on all pages, containing:
• Student Name / Logo
• Navigation menu with the following links:
• Home
• About
• Skills
• Contact
ABOUT ME PAGE
File Name: about.html
This page must contain:
• A brief personal introduction (minimum 4–5 lines)
• One profile image
SKILLS & PROJECTS PAGE
File Name: skills.html
Create a Skills/Projects section containing a minimum of threeEach card must include:
• Skill/Project Name
• 1–2 lines description
• Background color
• Border
All cards must be aligned using Flexbox.
CONTACT FORM PAGE
File Name: contact.html
Create a properly styled Contact Form with the following input fields:
• Name
• Email
• Subject
• Message
• Submit Button
